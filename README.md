# 南客松 S2 直通卡系统

MVP for collecting direct-pass records, generating QR codes, and managing internal notes.

## Getting Started

Install dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

Copy `.env.example` to `.env.local` and set:

- `DATABASE_URL`
- `APP_URL`
- `ADMIN_PASSWORD`
- `ADMIN_COOKIE_SECRET`
